About Our Project

This project is developed as part of SWE381 (Software Engineering) course. It's a Stack Overflow Like system that aims to provide a platform for developers to ask questions, share knowledge, and collaborate with others in the programming community.

Our platform features a modern, user-friendly interface with dark mode support, real-time interactions, and a comprehensive Q&A system that helps developers find solutions to their programming challenges.
